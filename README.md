## 📖 Overview

**LifeSaverQR** is a Django-based web platform that generates unique QR codes linked to encrypted emergency medical profiles. Designed for real-world emergencies, it gives first responders instant access to critical patient data — blood group, allergies, medications, and emergency contacts — by simply scanning a QR code.

> Built as part of my personal portfolio to solve a real-world problem in emergency healthcare response.

---

## ✨ Features

- 🔐 **Secure Authentication** — User registration, login, and session management
- 🧬 **Medical Profile Management** — Store blood group, allergies, medications, and conditions
- 📇 **Emergency Contacts** — Multiple contacts with priority levels
- 📲 **QR Code Generation** — Unique, downloadable QR codes linked to each profile
- 🔍 **QR Scan Access** — Instant emergency view without login required
- 📍 **Scan Tracking** — IP tracking and Geolocation API integration to log and alert users of scan locations via analytics dashboard
- 📄 **Medical Report Uploads** — PDF upload and management for medical history
- 📊 **User Dashboard** — Clean responsive interface for managing all profiles
